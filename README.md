Python (3.6 recommended)

Install required packages and libraries:
Open cv 4.1.1  
pip install opencv-python

Numpy
pip install numpy

Imutils
pip install imutils

Pytesseract
pip install pytesseract

https://github.com/tesseract-ocr/tesseract/wiki
